How to Download and Install Harran

1. Download the files using the link provided
2. Change the path in files, "vbs" and "bat" to where both files are located on your machine
3. Open "app.py" and scroll down until you the see the variables "Email" and "Password", replace the default values with your email and password
4. Place a shortcut of "vbs" file in startup folder.
5. Restart your machine.
